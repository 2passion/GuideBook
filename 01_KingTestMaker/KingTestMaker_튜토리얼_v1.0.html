STEP 1 / 5

기본 설정 — 문항 DB 폴더 지정

📸 기본 설정 화면
탐색 버튼으로 문항 DB 폴더 선택

1. 앱을 처음 실행하면 기본 설정 화면이 나타납니다.

2. 탐색 버튼을 클릭해 문항이 저장된 폴더를 선택합니다.

3. 저장 버튼을 클릭합니다.

✅ 저장 완료 메시지가 뜨면 다음 단계로 진행하세요.

STEP 2 / 5

문항 선택

📸 문항 목록 화면
좌측 단원 트리 + 우측 문항 체크리스트

1. 좌측 목록에서 단원·유형을 선택합니다.

2. 사용할 문항을 체크합니다.

3. 시험지에 추가 버튼을 클릭합니다.

✅ 선택한 문항이 시험지 목록에 추가되면 다음 단계로 진행하세요.
2 / 5
STEP 3 / 5

시험지 구성

📸 시험지 구성 화면
문항 순서 드래그, 배점 입력란

1. 추가된 문항의 순서를 드래그로 조정합니다.

2. 문항별 배점을 입력합니다.

3. 시험지 제목, 날짜, 반 정보를 입력합니다.

✅ 배점 합계가 목표 점수와 일치하는지 확인하세요.
3 / 5
STEP 4 / 5

미리보기

📸 미리보기 화면
시험지 / 정답지 탭 전환

1. 미리보기 버튼을 클릭해 최종 레이아웃을 확인합니다.

2. 정답지 탭을 눌러 정답지도 함께 확인합니다.

✅ 문항 순서와 배점이 올바른지 최종 확인하세요.
4 / 5
STEP 5 / 5

저장 및 완료

📸 저장 완료 화면

1. 이상이 없으면 저장 버튼을 클릭합니다.

2. 정답지는 시험지 저장 시 자동으로 함께 생성됩니다.

✅ 저장 완료! 아래 완료 화면에서 전체 요약을 확인하세요.
5 / 5
