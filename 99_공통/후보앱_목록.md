# GuideBook 후보 앱 목록

## 운영 원칙

- 폴더 있음 = 개발 중 (실제 프로젝트)
- 폴더 없음 = 후보 (이 문서로만 관리)

프로젝트 승인 시 구현계획서 → 작업지시서 → apps/{App}/ 생성 순서로 진행한다.
