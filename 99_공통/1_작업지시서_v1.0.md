  # PrintFlow 업무 매뉴얼 (SOP)
  대상: 신규 조교·강사 | 작성일: 2026-07 | 버전: v1.0

  ## 0. PrintFlow란?
  기존: 텍스트파일 → 조교가 폴더 직접 탐색 → 출력
  PrintFlow: 강사가 앱에서 자료선택·학생선택·계획등록 → 조교가 계획열람·출력·완료

  ## 1. 업무 흐름 한눈에 보기
  강사K → 자료선택 → 목록생성 → 학생선택 → 계획등록 → (동기화)
  → 조교P → 새로고침 → 계획열람 → 출력 → 완료
  → 조교Q/R → 미완료 이어서 처리

  ## A. 앱 설정
  ### A-1. 기본 설정
  - 사용자 이름 입력 후 저장 클릭
  - 자료 DB 폴더: 탐색 버튼으로 선택 후 반드시 저장 & 재스캔 클릭
  - 동기화 폴더: 탐색 버튼으로 공유 폴더 선택
  ⚠️ 자료 DB 변경 시 반드시 저장 & 재스캔 클릭 — 안 하면 자료 안 보임

  ### A-2. 학생 등록
  - 학생 관리 메뉴 → + 학생 추가 버튼
  - 이름, 반, 학교, 학년, 담당강사 입력

  ## B. 강사(K) 사용법
  ### B-1. 자료 열람 > 폴더 탐색
  - 좌측 트리에서 단원 선택 → 우측 파일 목록 확인
  - 필요한 파일 체크 → 목록에 추가 클릭

  ### B-2. 내 목록/계획 탭 — 목록 이름 지정
  - 내 목록/계획 탭 클릭
  - 저장된 목록 선택 → 이름 변경 버튼으로 명확한 이름 부여

  ### B-3. 계획 등록
  - 목록 선택 드롭다운에서 목록 선택
  - 반 선택 입력
  - 학생 선택 (N명 선택됨 확인)
  - 수업 예정일 날짜 선택
  - N명 × M개 파일 = Task 수 확인 후 계획 등록 (N개) 클릭

  ## C. 조교(P·Q·R) 사용법
  ### C-1. 출근 즉시 — 새로고침
  - 앱 실행 후 좌측 상단 새로고침 버튼 클릭
  - "아직 새로고침 전" → "새로고침 완료" 확인

  ### C-2. 계획 열람
  - 계획 열람 메뉴 클릭
  - 달력에서 오늘 날짜 클릭
  - 미완료(주황 배경) 항목 확인

  ### C-3. 출력 및 완료
  - 열기 버튼 → 파일 열기 → 학생 수만큼 출력
  - 출력 완료 후 반드시 완료 버튼 클릭
  ⚠️ 완료 클릭 안 하면 다음 조교가 중복 출력

  ## D. 조교 간 인수인계
  - 완료한 항목: 초록 배경으로 표시됨
  - 미완료 항목: 다음 조교가 새로고침 후 이어서 처리
  - 완료를 억지로 누르지 말 것 — 실제 출력 후에만 클릭

  ## E. 자주 하는 실수
  | 증상 | 원인 | 해결 |
  |------|------|------|
  | 자료가 안 보인다 | 저장 & 재스캔 미클릭 | 재스캔 실행 |
  | 강사 계획이 안 보인다 | 새로고침 미클릭 | 새로고침 클릭 |
  | 학생 선택이 안 된다 | 학생 관리에 미등록 | 학생 관리에서 등록 |
  | 계획이 등록 안 된다 | 학생 미선택 | 학생 선택 후 등록 |
  | 출력이 중복되었다 | 완료 버튼 미클릭 | 완료 처리 철저히 |
  | 다른 PC 변경사항 미반영 | 새로고침 미클릭 | 새로고침 클릭 |

  ## F. FAQ
  Q. 출근 후 가장 먼저 할 일은?
  A. 새로고침 → 계획 열람

  Q. 자료 DB 폴더를 바꿨는데 자료가 안 보여요?
  A. 기본 설정에서 저장 & 재스캔 클릭

  Q. 완료를 잘못 눌렀어요?
  A. 완료 ↩ 버튼으로 되돌리기 가능

  Q. 다른 조교가 처리한 내용이 안 보여요?
  A. 새로고침 클릭

# GuideBook Generator

학원 앱 가이드북(SOP + HTML 튜토리얼) 자동 생성 도구

GuideBook_Generator.bat 더블클릭

1단계: 가이드북 생성 / 앱 관리 / 설정 / 종료

4. 앱 이름 입력 → apps/{앱이름}/ 자동 생성
5. apps/{앱이름}/input/guide.md 파일 작성
6. 메인 메뉴에서 생성 실행

apps/{앱이름}/input/guide.md  ← 여기에 내용 작성
apps/{앱이름}/input/images/   ← 여기에 스크린샷 저장
apps/{앱이름}/output/         ← 여기에 결과물 생성

python --version  (3.8 이상 필요)
없으면: https://www.python.org/downloads/ 에서 설치

# GuideBook Generator - 생성 엔진
# Python 표준 라이브러리만 사용
# 인코딩: UTF-8

import sys
import os
import shutil
from datetime import datetime
from pathlib import Path

def get_base_dir():
    """실행 파일 기준 경로 반환"""
    return Path(__file__).parent

def backup_file(filepath):
    """기존 파일을 archive/ 폴더에 백업"""
    if not filepath.exists():
        return
    archive_dir = get_base_dir() / "archive"
    archive_dir.mkdir(exist_ok=True)
    timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
    backup_name = f"{filepath.stem}_{timestamp}{filepath.suffix}"
    shutil.copy2(filepath, archive_dir / backup_name)
    print(f"  [백업] {backup_name}")

def read_guide(app_name):
    """guide.md 읽기"""
    guide_path = get_base_dir() / "apps" / app_name / "input" / "guide.md"
    if not guide_path.exists():
        print(f"[오류] guide.md 파일을 찾을 수 없습니다.")
        print(f"  위치: {guide_path}")
        print(f"  해결: 위 경로에 guide.md 파일을 작성하세요.")
        sys.exit(1)
    return guide_path.read_text(encoding="utf-8")

def get_images(app_name):
    """images/ 폴더에서 이미지 파일 목록 반환 (파일명 순)"""
    images_dir = get_base_dir() / "apps" / app_name / "input" / "images"
    if not images_dir.exists():
        return []
    exts = {".png", ".jpg", ".jpeg", ".webp"}
    images = sorted([f for f in images_dir.iterdir() if f.suffix.lower() in exts])
    return images

def generate_sop(app_name, guide_content):
    """SOP 생성: .md + .html"""
    output_dir = get_base_dir() / "apps" / app_name / "output"
    output_dir.mkdir(parents=True, exist_ok=True)

    today = datetime.now().strftime("%Y-%m-%d")

    # --- SOP Markdown ---
    md_path = output_dir / f"{app_name}_SOP_v1.0.md"
    backup_file(md_path)
    md_content = f"# {app_name} 업무 매뉴얼 (SOP)\n"
    md_content += f"생성일: {today} | 버전: v1.0\n\n---\n\n"
    md_content += guide_content
    md_path.write_text(md_content, encoding="utf-8")
    print(f"  [생성] {md_path.name}")

    # --- SOP HTML ---
    html_path = output_dir / f"{app_name}_SOP_v1.0.html"
    backup_file(html_path)

    # guide.md의 마크다운을 간단히 HTML로 변환

def generate_tutorial(app_name, guide_content, images):
    """튜토리얼 HTML 생성"""
    output_dir = get_base_dir() / "apps" / app_name / "output"
    output_dir.mkdir(parents=True, exist_ok=True)

    today = datetime.now().strftime("%Y-%m-%d")

    # guide.md에서 STEP 파싱 (## STEP 또는 ## 으로 시작하는 섹션)
    sections = []
    current = None
    for line in guide_content.split("\n"):
        if line.startswith("## "):
            if current:
                sections.append(current)
            current = {"title": line[3:], "content": []}
        elif current:
            current["content"].append(line)
    if current:
        sections.append(current)

    if not sections:
        sections = [{"title": f"{app_name} 사용법", "content": guide_content.split("\n")}]

    # 이미지를 base64로 읽기 (상대경로 참조 대신)
    def img_tag(idx):
        if idx < len(images):
            import base64
            img_path = images[idx]
            with open(img_path, "rb") as f:
                b64 = base64.b64encode(f.read()).decode()
            ext = img_path.suffix[1:]
            return f'<img src="data:image/{ext};base64,{b64}" alt="화면 {idx+1}" style="max-width:100%;border-radius:8px;border:1px solid #e5e7eb;">'
        else:
            return f'<div class="mock-screen">📸 화면 {idx+1} 캡처 영역<br><small>images/ 폴더에 이미지를 추가하면 여기에 표시됩니다</small></div>'

    steps_html = ""
    for i, sec in enumerate(sections):
        content_html = ""
        for line in sec["content"]:
            if line.startswith("- "):
                content_html += f"<li>{line[2:]}</li>"
            elif line.startswith("> ⚠️"):
                content_html += f'<div class="warning">{line[2:]}</div>'
            elif line.strip():
                content_html += f"<p>{line}</p>"

        steps_html += f"""
    <div class="step" id="step-{i}" {"style='display:block'" if i == 0 else "style='display:none'"}>
      <div class="step-num">STEP {i+1} / {len(sections)}</div>
      <h2>{sec['title']}</h2>
      <div class="progress-bar"><div class="progress-fill" style="width:{int((i+1)/len(sections)*100)}%"></div></div>
      <div class="screen-area">{img_tag(i)}</div>
      <div class="content">{content_html}</div>
      <div class="checkpoint">✅ 이 화면이 보이면 다음 단계로 진행하세요.</div>
      <div class="nav">
        <button onclick="go({i-1})" {"disabled" if i==0 else ""}>← 이전</button>
        <span>{i+1} / {len(sections)}</span>
        <button onclick="go({i+1})" {"disabled" if i==len(sections)-1 else ""}>다음 →</button>
      </div>
    </div>"""

    html_path = output_dir / f"{app_name}_튜토리얼_v1.0.html"
    backup_file(html_path)
