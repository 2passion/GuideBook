## 작업 내용

`guide_KingTestMaker.md`에서 아래 두 섹션을 삭제한다.

**삭제 대상 1**

```markdown
## 자주 하는 실수
| 증상 | 원인 | 해결 |
|------|------|------|
| 복테 PDF가 안 만들어진다 | 학생 이름 미입력 | 이름 입력 후 재시도 |
| 단원 선택이 안 된다 | DB 폴더 미설정 | DB 폴더 경로 확인 |
| 오답 데이터가 없다 | JSON 불러오기 미실행 | JSON 불러오기 클릭 |
```

**삭제 대상 2**

```markdown
## FAQ
Q. DB 폴더는 어디로 설정하나요?
A. 좌측 하단 DB 폴더 항목에서 WANT DB 폴더로 지정합니다.

Q. 다른 PC의 오답 데이터를 가져오려면?
A. JSON 내보내기로 파일 저장 → 다른 PC에서 JSON 불러오기 클릭
```
