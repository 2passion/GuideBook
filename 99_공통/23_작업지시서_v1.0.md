# Generator archive Git 추적 제외

작성일: 2026-07-10
작업 대상: .gitignore
작업 성격: Git 운영 정리 (기능/콘텐츠 변경 없음)

---

## 작업 목표

Generator 실행 시 자동 생성되는 archive 백업 파일을
Git 추적 대상에서 제외한다.

GitHub는 버전 이력을 관리하고,
archive는 실행 직전 복구용 백업으로 역할을 분리한다.

---

## 역할 분리 원칙

```
GitHub  →  버전 관리 (commit, 영구 이력)
archive →  실행 직전 백업 (복구용, Git 제외)
```

---

## 작업 내용

### 1. .gitignore 수정
아래 내용을 .gitignore에 추가한다.

```
# Generator archive (실행 직전 자동 백업 — Git 제외)
GuideBook_생성기/archive/*
!GuideBook_생성기/archive/.gitkeep
```

결과:
- archive/ 폴더는 유지됨
- 폴더 안의 백업 파일은 Git 추적 제외
- .gitkeep 파일만 예외 허용 (폴더 존재 보장)

### 2. 이미 추적 중인 archive 파일 추적 해제

```bash
git rm -r --cached GuideBook_생성기/archive
```

주의: 로컬 파일은 삭제되지 않음. Git 추적만 해제됨.

### 3. commit

```bash
git add .gitignore
git commit -m "chore: ignore generator archive"
git push
```

---

## 작업 순서

1. .gitignore 파일 열기
2. archive 제외 규칙 추가
3. git rm -r --cached 실행 (추적 해제)
4. git add .gitignore
5. git commit -m "chore: ignore generator archive"
6. git push
7. git status 확인 (archive 파일 미추적 확인)

---

- [ ] commit 메시지: chore: ignore generator archive
