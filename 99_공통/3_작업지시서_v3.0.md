### 02_PrintFlow/ 생성

# PrintFlow 업무 매뉴얼 (SOP)
대상: 신규 조교·강사 | 작성일: 2026-07 | 버전: v1.0

## 0. PrintFlow란?
기존: 텍스트파일 → 조교가 폴더 직접 탐색 → 출력
PrintFlow: 강사가 앱에서 자료선택·학생선택·계획등록 → 조교가 계획열람·출력·완료

## 1. 업무 흐름 한눈에 보기
강사K → 자료선택 → 목록생성 → 학생선택 → 계획등록 → (동기화)
→ 조교P → 새로고침 → 계획열람 → 출력 → 완료
→ 조교Q/R → 미완료 이어서 처리

## A. 앱 설정
### A-1. 기본 설정
- 사용자 이름 입력 후 저장 클릭
- 자료 DB 폴더: 탐색 버튼으로 선택 후 반드시 저장 & 재스캔 클릭
- 동기화 폴더: 탐색 버튼으로 공유 폴더 선택
⚠️ 자료 DB 변경 시 반드시 저장 & 재스캔 클릭 — 안 하면 자료 안 보임

### A-2. 학생 등록
- 학생 관리 메뉴 → + 학생 추가 버튼
- 이름, 반, 학교, 학년, 담당강사 입력

## B. 강사(K) 사용법
### B-1. 자료 열람 > 폴더 탐색
- 좌측 트리에서 단원 선택 → 우측 파일 목록 확인
- 필요한 파일 체크 → 목록에 추가 클릭

### B-2. 내 목록/계획 탭
- 내 목록/계획 탭 클릭
- 저장된 목록 선택 → 이름 변경 버튼으로 명확한 이름 부여

### B-3. 계획 등록
- 목록 선택 드롭다운에서 목록 선택
- 반 선택 입력
- 학생 선택 (N명 선택됨 확인)
- 수업 예정일 날짜 선택
- N명 × M개 파일 = Task 수 확인 후 계획 등록 (N개) 클릭

## C. 조교(P·Q·R) 사용법
### C-1. 출근 즉시 — 새로고침
- 앱 실행 후 좌측 상단 새로고침 버튼 클릭
- "아직 새로고침 전" → "새로고침 완료" 확인

### C-2. 계획 열람
- 계획 열람 메뉴 클릭
- 달력에서 오늘 날짜 클릭
- 미완료(주황 배경) 항목 확인

### C-3. 출력 및 완료
- 열기 버튼 → 파일 열기 → 학생 수만큼 출력
- 출력 완료 후 반드시 완료 버튼 클릭
⚠️ 완료 클릭 안 하면 다음 조교가 중복 출력

## D. 조교 간 인수인계
- 완료한 항목: 초록 배경으로 표시됨
- 미완료 항목: 다음 조교가 새로고침 후 이어서 처리

## E. 자주 하는 실수
| 증상 | 원인 | 해결 |
|------|------|------|
| 자료가 안 보인다 | 저장 & 재스캔 미클릭 | 재스캔 실행 |
| 강사 계획이 안 보인다 | 새로고침 미클릭 | 새로고침 클릭 |
| 학생 선택이 안 된다 | 학생 관리에 미등록 | 학생 관리에서 등록 |
| 계획이 등록 안 된다 | 학생 미선택 | 학생 선택 후 등록 |
| 출력이 중복되었다 | 완료 버튼 미클릭 | 완료 처리 철저히 |
| 다른 PC 변경사항 미반영 | 새로고침 미클릭 | 새로고침 클릭 |

## F. FAQ
Q. 출근 후 가장 먼저 할 일은?
A. 새로고침 → 계획 열람

Q. 자료 DB 폴더를 바꿨는데 자료가 안 보여요?
A. 기본 설정에서 저장 & 재스캔 클릭

Q. 완료를 잘못 눌렀어요?
A. 완료 ↩ 버튼으로 되돌리기 가능

Q. 다른 조교가 처리한 내용이 안 보여요?
A. 새로고침 클릭

### 파일 3: generator.py

# GuideBook Generator — 생성 엔진
# guide.md → SOP(.md + .html) + 튜토리얼(.html) 변환
# Python 표준 라이브러리만 사용

import sys
import os
import shutil
from datetime import datetime
from pathlib import Path

def get_base_dir():
    """실행 파일 기준 경로 반환"""
    return Path(__file__).parent

def backup_file(filepath):
    """기존 파일을 archive/에 백업"""
    if not filepath.exists():
        return
    archive_dir = get_base_dir() / "archive"
    archive_dir.mkdir(exist_ok=True)
    timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
    backup_name = f"{filepath.stem}_{timestamp}{filepath.suffix}"
    shutil.copy2(filepath, archive_dir / backup_name)
    print(f"  [백업] {backup_name}")

def read_guide(app_name):
    """apps/{앱}/input/guide.md 읽기"""
    guide_path = get_base_dir() / "apps" / app_name / "input" / "guide.md"
    if not guide_path.exists():

# GuideBook Generator — 소스 분석 엔진
# 실제 프로젝트 소스 폴더를 읽어 guide.md 자동 생성
# Python 표준 라이브러리만 사용

import sys
import os
from datetime import datetime
from pathlib import Path

# 읽기 지원 확장자 및 우선순위
PRIORITY_EXTS = [".md", ".txt", ".html", ".jsx", ".js", ".py", ".bat", ".sql", ".css", ".json"]
MAX_LINES = 500  # 파일 1개당 최대 읽기 줄 수

def get_base_dir():
    return Path(__file__).parent

def read_file_safe(filepath):
    """파일을 안전하게 읽기 (실패 시 None 반환)"""
    for enc in ["utf-8", "cp949", "utf-8-sig"]:
        try:
            lines = filepath.read_text(encoding=enc).splitlines()
            return "\n".join(lines[:MAX_LINES])
        except Exception:
            continue
    return None

def scan_folders(src_root, folder_list):
    """지정 폴더들의 파일 목록을 우선순위 순으로 반환"""
    src_path = Path(src_root)
    all_files = []
    for folder in folder_list:
        folder_path = src_path / folder.strip()
        if not folder_path.exists():
            print(f"  [건너뜀] 폴더 없음: {folder_path}")
            continue
        for ext in PRIORITY_EXTS:
            files = sorted(folder_path.rglob(f"*{ext}"))
            all_files.extend(files)
    return all_files

def extract_content(files):
    """파일 목록에서 내용 추출"""
    overview_lines = []   # 앱 개요 (README, docs의 md)
    ui_lines = []         # 화면/메뉴 (html, jsx)
    procedure_lines = []  # 업무 절차 (command, docs의 md)
    error_lines = []      # 오류/주의 (log)
    read_files = []

    for f in files:
        content = read_file_safe(f)
        if not content:
            continue

        print(f"  [읽기] {f.name}")
        read_files.append(str(f))
        parts = f.parts
        folder_name = parts[-2] if len(parts) >= 2 else ""

        # 폴더별 분류
        if folder_name.lower() in ("docs", "readme") or f.name.lower() == "readme.md":
            overview_lines.append(f"\n### {f.name}\n{content[:800]}")
        elif folder_name.lower() in ("log", "archive"):
            error_lines.append(f"\n### {f.name}\n{content[:400]}")
        elif folder_name.lower() in ("command",):
            procedure_lines.append(f"\n### {f.name}\n{content[:600]}")
        elif f.suffix.lower() in (".html", ".jsx"):
            ui_lines.append(f"\n### {f.name}\n{content[:400]}")
        elif f.suffix.lower() in (".md",):
            procedure_lines.append(f"\n### {f.name}\n{content[:500]}")
        else:
            procedure_lines.append(f"\n### {f.name}\n{content[:300]}")

    return overview_lines, ui_lines, procedure_lines, error_lines, read_files

def generate_guide_md(app_name, src_root, folder_list_str, output_path):
    """분석 결과로 guide.md 생성"""
    folder_list = [f.strip() for f in folder_list_str.split(",")]
    today = datetime.now().strftime("%Y-%m-%d")

    print(f"\n[분석 시작] {app_name}")
    print(f"  소스: {src_root}")
    print(f"  폴더: {', '.join(folder_list)}")

    files = scan_folders(src_root, folder_list)
    print(f"\n  총 {len(files)}개 파일 발견")

    overview, ui, procedure, errors, read_files = extract_content(files)

    guide = f"""# {app_name} 가이드
생성일: {today}
소스 경로: {src_root}
분석 폴더: {', '.join(folder_list)}
분석 파일 수: {len(read_files)}개

---

## 앱 개요
{''.join(overview) if overview else '(docs/ 또는 README.md에서 자동 추출됩니다)'}

---

## 주요 화면 및 메뉴
{''.join(ui) if ui else '(html/jsx 파일에서 자동 추출됩니다)'}

---

## 업무 절차
{''.join(procedure) if procedure else '(command/ 또는 docs/ 파일에서 자동 추출됩니다)'}

---

## 주의사항 및 오류
{''.join(errors) if errors else '(log/ 파일에서 자동 추출됩니다)'}

---

## 자주 하는 실수
| 증상 | 원인 | 해결 |
|------|------|------|
| (log 분석 후 추가 예정) | | |

---

## FAQ
Q. (docs 분석 후 추가 예정)
A.

---

## 분석 파일 목록 ({len(read_files)}개)
{'chr(10).join(f"- {f}" for f in read_files)}
"""

    output_path = Path(output_path)
    output_path.parent.mkdir(parents=True, exist_ok=True)
    output_path.write_text(guide, encoding="utf-8")
    print(f"\n[완료] guide.md 생성: {output_path}")
    return str(output_path)

def main():

if __name__ == "__main__":
