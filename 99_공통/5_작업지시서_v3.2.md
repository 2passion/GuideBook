## 수정 내용 2 — generate_tutorial() 헤더에 PDF 버튼 추가

### 변경 목표
- 기존: 헤더 우측 🖨️ 인쇄 버튼 1개
- 변경: 헤더 우측 🖨️ 인쇄 + 📄 PDF 버튼 2개 나란히

### 헤더 버튼 영역 변경

generate_tutorial() 안에서 헤더 버튼 부분을 찾아 교체:

기존 (인쇄 버튼):
    <button onclick="window.print()"
            style="padding:6px 14px;background:white;
                   border:1px solid #d1d5db;border-radius:6px;
                   cursor:pointer;font-size:13px;">인쇄</button>

변경 후 (인쇄 + PDF 버튼 묶음):
    <div style="display:flex;gap:8px;">
      <button onclick="window.print()"
              style="padding:6px 14px;background:white;
                     border:1px solid #d1d5db;border-radius:6px;
                     cursor:pointer;font-size:13px;font-family:inherit;">
        🖨️ 인쇄
      </button>
      <button onclick="tutorialPDF()"
              style="padding:6px 14px;background:#16a34a;color:white;
                     border:none;border-radius:6px;cursor:pointer;
                     font-size:13px;font-family:inherit;">
        📄 PDF
      </button>
    </div>

### role_js 변수 끝에 tutorialPDF 함수 추가

has_roles == True 일 때의 role_js:
기존 마지막 줄 다음에 추가:
function tutorialPDF() {{
  var orig = document.title;
  document.title = '{app_name}_튜토리얼_v1.0';
  window.print();
  setTimeout(function() {{ document.title = orig; }}, 1000);
}}

has_roles == False 일 때의 role_js:
기존 마지막 줄 다음에 추가:
function tutorialPDF() {{
  var orig = document.title;
  document.title = '{app_name}_튜토리얼_v1.0';
  window.print();
  setTimeout(function() {{ document.title = orig; }}, 1000);
}}

### @media print 스타일 추가 (튜토리얼 CSS 안)

기존:
  @media print{{.nav,.role-tabs{{display:none}}}}

변경 후:
  @media print{{
    .nav,.role-tabs{{display:none}}
    div[style*="gap:8px"]{{display:none}}
  }}

## PDF 저장 방법 (사용자 안내)

1. **📄 PDF 저장** 버튼 클릭
2. 인쇄 창 → 프린터 목록에서 **Microsoft Print to PDF** 선택
3. **인쇄** 클릭 → 저장 위치와 파일명 지정
   (파일명이 자동으로 입력됨)
