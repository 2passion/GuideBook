## STEP 2. print CSS 수정

`generate_sop()` 함수의 `@media print` 블록을 아래로 교체한다.

### 변경 전
```css
@media print{
  .sticky-header{position:static}
  .top-bar{border-bottom:none}
  .btn-group{display:none}
  .pdf-tip{display:none}
  body{padding:0}
}
```

### 변경 후
```css
@media print{
  /* 레이아웃 */
  .sticky-header{position:static}
  .top-bar{border-bottom:none}
  body{padding:0}

  /* 버튼·안내 숨김 */
  .btn-group{display:none}
  .pdf-tip{display:none}
  .tab-bar{display:none}

  /* 탭 전체 출력 */
  .tab-panel{display:block !important}

  /* 탭 간 구분선 */
  .tab-panel + .tab-panel{
    border-top:2px solid #e5e7eb;
    margin-top:32px;
    padding-top:32px;
  }

  /* 페이지 나누기 — 탭별로 새 페이지 (선택) */
  /* .tab-panel{page-break-before:always} */
}
```

> ⚠️ `page-break-before:always`는 기본 주석 처리.
> 탭별 새 페이지가 필요하면 주석 해제.

## STEP 8. 오류 발생 시 처리

자주 발생하는 오류:
- CSS f-string 이스케이프: `{` → `{{`, `}` → `}}` 확인
- `.tab-panel + .tab-panel` CSS 선택자: Python 문자열 내 `+` 특수문자 이스케이프 불필요
- 인쇄 미리보기에서 탭 내용 미표시: `display:block !important` 우선순위 확인
- 탭 전환 JS 미동작: print 후 브라우저 새로고침 필요할 수 있음 (정상)
