# PrintFlow 코드 체계 통일

작성일: 2026-07-10
작업 대상: reference_PrintFlow.md / rule_PrintFlow.md
작업 성격: 콘텐츠 리팩터링 (기능 변경 없음)

---

## 작업 목표

이번 작업은 코드 체계 통일을 위한 리팩터링이다.

구 코드(01~09, a~m)를 신 코드(A/B/C/D, grade/step, Mark-A/B)로 교체한다.
내용을 삭제하는 것이 아니라, 코드 체계를 통일한다.

Generator 기능은 수정하지 않는다.

| f | 블랙라벨 | 중등 |

| h | 쎈 | 고등 |

| k | 고쟁이 | 고등 |
| m | 자이스토리 | 고등 |

  step-a 개념서 / step-b 유형서 / step-c 심화서
  step-d 모의고사 / step-e 기출

| 코드 | 교재명 | 구분 |
| grade-a_step-a | 기본+응용 | 초등 개념서 |
| grade-a_step-b | 쎈 | 초등 유형서 |
| grade-a_step-c | 최상위S | 초등 심화서 |
| grade-b_step-a | 개념+유형 | 중등 개념서 |
| grade-b_step-b | 쎈 | 중등 유형서 |
| grade-b_step-c | 일품 | 중등 심화서 |
| grade-b_step-c | 블랙라벨 | 중등 심화서 |
| grade-b_step-d | 내신모의 | 중등 모의고사 |
| grade-b_step-e | 기출 | 중등 기출 |
| grade-c_step-a | 개념원리 | 고등 개념서 |
| grade-c_step-b | 쎈 | 고등 유형서 |
| grade-c_step-b | 마플시너지 | 고등 유형서 |
| grade-c_step-c | 일품 | 고등 심화서 |

| grade-c_step-c | 고쟁이 | 고등 심화서 |
| grade-c_step-c | 블랙라벨 | 고등 심화서 |

### 4. rule_PrintFlow.md — 검사 규칙 교체

#### Before (구 코드)
```
A. 숙제 검사 대상
  A-01 교재 검사
  A-02 복테 검사
  A-03 오답노트 검사
  A-04 숙제 프린트 검사
  A-06 내신모의 검사
  A-07 기출 검사

B. 테스트 검사 대상
  B-05 테스트 프린트 검사
  B-06 내신모의 검사
  B-07 기출 검사
  B-08 진단평가 검사
  B-09 입학테스트 검사
```

#### After (신 코드)
```
Mark-A. 숙제 검사 대상
  Mark-A_A-01 교재 검사
  Mark-A_A-02 복테 검사
  Mark-A_A-03 오답노트 검사
  Mark-A_B-01 숙제 프린트 검사
  Mark-A_C-01 내신모의 검사
  Mark-A_C-02 기출 검사

Mark-B. 테스트 검사 대상
  Mark-B_B-02 테스트 프린트 검사
  Mark-B_C-01 내신모의 검사
  Mark-B_C-02 기출 검사
  Mark-B_D-01 진단평가 검사
  Mark-B_D-02 입학테스트 검사
```
