KingTestMaker 가이드

생성일: 2026-07 | 버전: v1.0


앱 개요

복습테스트(복테) PDF + 오답노트 PDF 자동 생성 앱 (EXE)

주 사용자: 조교


업무 흐름

복테 제작 → JSON 내보내기 → 다른 PC 전달 → JSON 불러오기 → 오답노트 PDF 생성


복테 제작 전 확인

□ 학생 이름 입력

□ 과정 선택

□ 교재 선택

□ 단원 선택

□ DB 폴더 확인

□ 저장 폴더 확인


업무① 복테 제작

(복테 제작 탭)

  • 학생 이름 입력
  • 과정 → 교재 → 단원 순서대로 선택
  • 페이지 양식 선택 (양식1~5)
  • 복테 PDF 제작 버튼 클릭

업무② 오답노트 제작

(오답기록표 관리 탭)

  • 좌측에서 학생 선택 (전체/고등부/중등부/초등부)
  • JSON 내보내기: 오답 데이터 파일로 저장 → 다른 PC에 전달
  • JSON 불러오기: 전달받은 파일 불러오기
  • 날짜 선택 → 오답노트 PDF 생성


KingTestMaker 업무 규칙

생성일: 2026-07 | 버전: v1.0


복테 제작 규칙

  • 학생 이름을 입력합니다.
  • 과정 → 교재 → 단원 순서로 선택합니다.
  • DB 폴더는 WANT DB 폴더로 지정합니다.
  • 저장 폴더는 작업 전 반드시 확인합니다.

JSON 전달 규칙

  • JSON 내보내기는 오답기록표 관리 탭에서 실행합니다.
  • JSON 파일을 다른 PC에 전달한 후 즉시 JSON 불러오기를 실행합니다.
  • JSON 불러오기 없이 오답노트 PDF를 생성하지 않습니다.

오답노트 제작 규칙

  • 날짜를 선택한 후 오답노트 PDF를 생성합니다.
  • 학생 선택은 전체/고등부/중등부/초등부 중 정확히 선택합니다.

FAQ

Q. DB 폴더는 어디로 설정하나요?

A. 좌측 하단 DB 폴더 항목에서 WANT DB 폴더로 지정합니다.


Q. 다른 PC의 오답 데이터를 가져오려면?

A. JSON 내보내기로 파일 저장 → 다른 PC에서 JSON 불러오기 클릭
