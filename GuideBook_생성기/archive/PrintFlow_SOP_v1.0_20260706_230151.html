PrintFlow 가이드

생성일: 2026-07 | 버전: v1.0


앱 개요

기존: 텍스트파일 → 조교가 폴더 직접 탐색 → 출력

PrintFlow: 강사가 앱에서 자료선택·학생선택·계획등록 → 조교가 계획열람·출력·완료


업무 흐름

강사K → 자료선택 → 목록생성 → 학생선택 → 계획등록 → (동기화)

→ 조교P → 새로고침 → 계획열람 → 출력 → 완료

→ 조교Q/R → 미완료 이어서 처리


기본 설정

  • 사용자 이름 입력 후 저장 클릭
  • 자료 DB 폴더: 탐색 버튼으로 선택 후 반드시 저장 & 재스캔 클릭
  • 동기화 폴더: 탐색 버튼으로 공유 폴더 선택
⚠️ 자료 DB 변경 시 반드시 저장 & 재스캔 클릭 — 안 하면 자료 안 보임

학생 등록

  • 학생 관리 메뉴 → + 학생 추가 버튼
  • 이름, 반, 학교, 학년, 담당강사 입력

강사(K) 사용법 — 자료 열람 > 폴더 탐색

  • 좌측 트리에서 단원 선택 → 우측 파일 목록 확인
  • 필요한 파일 체크 → 목록에 추가 클릭

강사(K) 사용법 — 내 목록/계획 탭

  • 내 목록/계획 탭 클릭
  • 저장된 목록 선택 → 이름 변경 버튼으로 명확한 이름 부여

강사(K) 사용법 — 계획 등록

  • 목록 선택 드롭다운에서 목록 선택
  • 반 선택 입력
  • 학생 선택 (N명 선택됨 확인)
  • 수업 예정일 날짜 선택
  • N명 × M개 파일 = Task 수 확인 후 계획 등록 (N개) 클릭

조교(P·Q·R) 사용법 — 출근 즉시 새로고침

  • 앱 실행 후 좌측 상단 새로고침 버튼 클릭
  • "아직 새로고침 전" → "새로고침 완료" 확인

조교(P·Q·R) 사용법 — 계획 열람

  • 계획 열람 메뉴 클릭
  • 달력에서 오늘 날짜 클릭
  • 미완료(주황 배경) 항목 확인

조교(P·Q·R) 사용법 — 출력 및 완료

  • 열기 버튼 → 파일 열기 → 학생 수만큼 출력
  • 출력 완료 후 반드시 완료 버튼 클릭
⚠️ 완료 클릭 안 하면 다음 조교가 중복 출력

조교 간 인수인계

  • 완료한 항목: 초록 배경으로 표시됨
  • 미완료 항목: 다음 조교가 새로고침 후 이어서 처리

FAQ

Q. 출근 후 가장 먼저 할 일은?

A. 새로고침 → 계획 열람


Q. 자료 DB 폴더를 바꿨는데 자료가 안 보여요?

A. 기본 설정에서 저장 & 재스캔 클릭


Q. 완료를 잘못 눌렀어요?

A. 완료 ↩ 버튼으로 되돌리기 가능


Q. 다른 조교가 처리한 내용이 안 보여요?

A. 새로고침 클릭

파일 저장 원칙 (8가지)


1. 빠짐없이 저장한다

2. 중복 없이 저장한다

3. 원본 파일은 유지하고 덮어쓰지 않는다

4. 시간 순서대로 저장한다

5. 파일을 열어보지 않아도 내용을 알 수 있도록 저장한다

6. 나중에 검색하기 쉽게 저장한다

7. 파일 출처를 알 수 있게 저장한다

8. 숙제와 테스트를 명확히 구분하여 저장한다


---


스캔 후 파일명 규칙


A. 숙제

규칙: `원본파일명 + 학생이름 + -횟수`


숙제의 횟수는 같은 자료를 학생이 몇 번째 푼 것인지를 나타낸다.


예시의미
A 김철수-1.pdfA 자료, 김철수, 첫 번째 숙제
A 김철수-2.pdfA 자료, 김철수, 두 번째 숙제
B 김철수-1.pdfB 자료, 김철수, 첫 번째 숙제

B. 테스트

규칙: `원본파일명 + 학생이름 + 틀린문제수 + 개`


테스트의 숫자는 시험에서 처음 틀린 문제의 전체 개수를 나타낸다.


예시의미
C 김철수 2개.pdfC 자료, 김철수, 최초 시험에서 2문제 틀림
D 김철수 3개.pdfD 자료, 김철수, 최초 시험에서 3문제 틀림

원본 파일 예시 (4개인 경우)

```

A.hwp B.pdf C.pdf D.hwp

```

스캔 후 저장:

```

A 김철수-1.pdf (숙제 첫 번째)

A 김철수-2.pdf (숙제 두 번째)

B 김철수-1.pdf (숙제 첫 번째)

C 김철수 2개.pdf (테스트, 2문제 틀림)

D 김철수 3개.pdf (테스트, 3문제 틀림)

```

검사 규칙


검사는 숙제 검사(A)와 테스트 검사(B) 두 가지로 구분한다.


A. 숙제 검사 대상

```

A-01 교재 검사

A-02 복테 검사

A-03 오답노트 검사

A-04 숙제 프린트 검사

A-06 내신모의 검사

A-07 기출 검사

```


B. 테스트 검사 대상

```

B-05 테스트 프린트 검사

B-06 내신모의 검사

B-07 기출 검사

B-08 진단평가 검사

B-09 입학테스트 검사

```

FAQ


Q. 자료 DB 폴더를 바꿨는데 자료가 그대로예요.

A. 폴더 변경 후 반드시 **저장 & 재스캔** 버튼을 클릭해야 합니다.


Q. 강사가 계획을 등록했는데 내 화면에 안 나와요.

A. 좌측 상단 **새로고침** 버튼을 클릭하면 최신 계획이 불러와집니다.


Q. 완료 버튼을 실수로 눌렀어요.

A. **완료 ↩** 버튼을 다시 누르면 미완료 상태로 되돌릴 수 있습니다.


Q. 계획 등록 시 학생이 목록에 없어요.

A. **학생 관리** 메뉴에서 먼저 학생을 등록해야 합니다.
