# KingTestMaker 가이드
생성일: 2026-07 | 버전: v1.0

## 앱 개요
복습테스트(복테) PDF + 오답노트 PDF 자동 생성 앱 (EXE)
주 사용자: 조교

## 업무 흐름
복테 제작 → JSON 내보내기 → 다른 PC 전달 → JSON 불러오기 → 오답노트 PDF 생성

## 복테 제작 전 확인
□ 학생 이름 입력
□ 과정 선택
□ 교재 선택
□ 단원 선택
□ DB 폴더 확인
□ 저장 폴더 확인

## 업무① 복테 제작
(복테 제작 탭)
- 학생 이름 입력
- 과정 → 교재 → 단원 순서대로 선택
- 페이지 양식 선택 (양식1~5)
- 복테 PDF 제작 버튼 클릭

## 업무② 오답노트 제작
(오답기록표 관리 탭)
- 좌측에서 학생 선택 (전체/고등부/중등부/초등부)
- JSON 내보내기: 오답 데이터 파일로 저장 → 다른 PC에 전달
- JSON 불러오기: 전달받은 파일 불러오기
- 날짜 선택 → 오답노트 PDF 생성
