# Dooly 가이드
생성일: 2026-07 | 버전: v1.0

## 앱 개요
학원 AI 업무 도우미 PWA (KING Assistant OS)
강사: 업무 지시 / 조교: 업무 실행

## 업무 흐름
강사 → 업무 추가 → JSON 다운로드 → (전달) → 조교 → JSON 업로드 → 업무 실행

## 업무① 업무 받기
(업무 탭 → JSON 업로드)
- 강사가 전달한 JSON 파일을 JSON 업로드로 불러오기
- 업무 목록 확인 (우선순위/진행상황/담당자 필터)
> ⚠️ 업무 목록이 비어 있으면 JSON 업로드를 먼저 실행

## 업무② 업무 실행
(업무 탭)
- 업무 선택 → 진행 → 완료 처리
- 오늘 할 일: 홈 탭 달력에서 날짜 클릭 → 업무 확인

## 업무③ 모르면 SOP
(SOP 탭)
- 업무 절차가 궁금할 때
- 카테고리 필터: 복테/오답노트/채점/비품/장비

## 업무⑤ 그래도 모르겠으면 AI
(Dooly 탭)
- 기본: Mock 모드 (오프라인, 사전 등록 답변)
- 온라인: Gemini AI 모드 (실시간 AI 답변)
- SOP/FAQ 참조 태그 자동 표시

## 참고 — 설정
(설정 탭, 최초 1회)
- 담당자 추가/관리
- DB 폴더 경로 설정

# Dooly 업무 규칙

## JSON 규칙

- JSON 파일은 덮어쓰지 않고 날짜별로 보관합니다.

## 업무 처리 규칙

- 업무는 반드시 Dooly 앱에서 완료 처리합니다.
- 업무 목록이 비어 있으면 JSON 업로드를 먼저 실행합니다.

## AI 챗봇 사용 규칙

- 기본 모드는 Mock 모드(오프라인)입니다.
- 실시간 AI 답변이 필요할 때만 Gemini 모드로 전환합니다.

## 설정 규칙

- 담당자 추가/관리는 설정 탭에서 최초 1회 실행합니다.
- DB 폴더 경로는 최초 설정 후 임의로 변경하지 않습니다.

# Dooly FAQ

## 업무④ 안 되면 FAQ

(FAQ 탭)
- 특정 상황 해결책이 궁금할 때
- 카테고리 필터: 복테/오답노트/채점/비품/제본기

## 자주 하는 실수

| 증상 | 원인 | 해결 |
|------|------|------|
| 업무가 안 보인다 | JSON 업로드 미실행 | JSON 업로드 클릭 |
| 동기화가 안 된다 | 오프라인 상태 | 네트워크 확인 후 동기화 |
| AI 답변이 없다 | Mock 모드 상태 | Dooly 탭에서 모드 확인 |

## FAQ

Q. 강사가 등록한 업무를 어떻게 받나요?
A. 강사가 JSON 다운로드로 저장한 파일을 받아 JSON 업로드로 불러옵니다.

Q. 언제 SOP를 보고, 언제 FAQ를 보나요?
A. 업무 절차가 궁금하면 SOP, 특정 상황의 해결책이 궁금하면 FAQ를 봅니다.

Q. AI 챗봇이 틀린 답을 줘요.
A. Mock 모드는 사전 등록 답변만 가능합니다. Gemini 모드로 전환하세요.
