# PrintFlow 참조 코드표

## 자료 분류 코드

### 대분류

| 코드 | 대분류 |
|------|--------|
| A | 교재(Book) |
| B | 프린트(Print) |
| C | 시험대비(Exam) |
| D | 성취도(Level) |

### 코드표

| 코드 | 용도 |
|------|------|
| A-01 | 교재 원본 자료 |
| A-02 | 복습 테스트(복테) |

---

## 폴더 구조 표준

자료는 아래 4단계 계층 구조로 관리한다.

```
종류 → 과정 → 교재 → 단원
```

### 과정 코드

#### 초등수학
```
1-01 초1-1 / 1-02 초1-2 / 1-03 초2-1 / 1-04 초2-2
1-05 초3-1 / 1-06 초3-2 / 1-07 초4-1 / 1-08 초4-2
1-09 초5-1 / 1-10 초5-2 / 1-11 초6-1 / 1-12 초6-2
```

#### 중등수학
```
2-01 중1-1 / 2-02 중1-2 / 2-03 중2-1
2-04 중2-2 / 2-05 중3-1 / 2-06 중3-2
```

#### 고등수학
```
3-01 공통수학1 / 3-02 공통수학2 / 3-03 대수
3-04 확률과 통계 / 3-05 미적분1 / 3-06 미적분2 / 3-07 기하
```

### 교재 코드

#### 학년별 grade
```
grade-a 초등 / grade-b 중등 / grade-c 고등
```

#### 단계별 step
```
step-a 개념서 / step-b 유형서 / step-c 심화서
step-d 모의고사 / step-e 기출
```
