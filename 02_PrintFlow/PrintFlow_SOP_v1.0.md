# PrintFlow 업무 매뉴얼 (SOP)
대상: 신규 조교·강사 | 작성일: 2026-07 | 버전: v1.0

---

## 0. PrintFlow란?

기존: 텍스트파일 → 조교가 폴더 직접 탐색 → 출력
PrintFlow: 강사가 앱에서 자료선택·학생선택·계획등록 → 조교가 계획열람·출력·완료

---

## 1. 업무 흐름 한눈에 보기

```
강사K → 자료선택 → 목록생성 → 학생선택 → 계획등록 → (동기화)
→ 조교P → 새로고침 → 계획열람 → 출력 → 완료
→ 조교Q/R → 미완료 이어서 처리
```

---

## A. 앱 설정

### A-1. 기본 설정
1. **사용자 이름**을 입력한 후 **저장** 버튼을 클릭합니다.
2. **자료 DB 폴더**: 탐색 버튼으로 선택한 후 반드시 **저장 & 재스캔** 버튼을 클릭합니다.
3. **동기화 폴더**: 탐색 버튼으로 공유 폴더를 선택합니다.

> ⚠️ 자료 DB 변경 시 반드시 **저장 & 재스캔**을 클릭하세요 — 안 하면 자료가 안 보입니다.

### A-2. 학생 등록
1. **학생 관리** 메뉴 → **+ 학생 추가** 버튼을 클릭합니다.
2. 이름, 반, 학교, 학년, 담당강사를 입력합니다.

---

## B. 강사(K) 사용법

### B-1. 자료 열람 > 폴더 탐색
1. 좌측 트리에서 단원을 선택하면 우측에 파일 목록이 표시됩니다.
2. 필요한 파일을 체크한 후 **목록에 추가** 버튼을 클릭합니다.

### B-2. 내 목록/계획 탭
1. **내 목록/계획** 탭을 클릭합니다.
2. 저장된 목록을 선택하고 **이름 변경** 버튼으로 명확한 이름을 부여합니다.

### B-3. 계획 등록
1. **목록 선택** 드롭다운에서 목록을 선택합니다.
2. **반 선택**을 입력합니다.
3. **학생 선택** (N명 선택됨을 확인합니다).
4. **수업 예정일** 날짜를 선택합니다.
5. N명 × M개 파일 = Task 수를 확인한 후 **계획 등록 (N개)** 버튼을 클릭합니다.

---

## C. 조교(P·Q·R) 사용법

### C-1. 출근 즉시 — 새로고침
1. 앱 실행 후 좌측 상단 **새로고침** 버튼을 클릭합니다.
2. "아직 새로고침 전" → "새로고침 완료"로 바뀌는지 확인합니다.

### C-2. 계획 열람
1. **계획 열람** 메뉴를 클릭합니다.
2. 달력에서 오늘 날짜를 클릭합니다.
3. 미완료(주황 배경) 항목을 확인합니다.

### C-3. 출력 및 완료
1. **열기** 버튼 → 파일 열기 → 학생 수만큼 출력합니다.
2. 출력 완료 후 반드시 **완료** 버튼을 클릭합니다.

> ⚠️ 완료 클릭 안 하면 다음 조교가 중복 출력하게 됩니다.

---

## D. 조교 간 인수인계

- 완료한 항목: 초록 배경으로 표시됩니다.
- 미완료 항목: 다음 조교가 새로고침 후 이어서 처리합니다.

---

## E. 자주 하는 실수

| 증상 | 원인 | 해결 |
|------|------|------|
| 자료가 안 보인다 | 저장 & 재스캔 미클릭 | 재스캔 실행 |
| 강사 계획이 안 보인다 | 새로고침 미클릭 | 새로고침 클릭 |
| 학생 선택이 안 된다 | 학생 관리에 미등록 | 학생 관리에서 등록 |
| 계획이 등록 안 된다 | 학생 미선택 | 학생 선택 후 등록 |
| 출력이 중복되었다 | 완료 버튼 미클릭 | 완료 처리 철저히 |
| 다른 PC 변경사항 미반영 | 새로고침 미클릭 | 새로고침 클릭 |

## F. FAQ

Q. 출근 후 가장 먼저 할 일은?
A. 새로고침 → 계획 열람

Q. 자료 DB 폴더를 바꿨는데 자료가 안 보여요?
A. 기본 설정에서 저장 & 재스캔 클릭

Q. 완료를 잘못 눌렀어요?
A. 완료 ↩ 버튼으로 되돌리기 가능

Q. 다른 조교가 처리한 내용이 안 보여요?
A. 새로고침 클릭
