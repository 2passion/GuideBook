STEP 3 / 6

오늘 날짜 클릭 → 미완료 항목 확인

1. 달력에서 오늘 날짜를 클릭합니다.

2. 미완료(주황 배경) 항목을 확인합니다.

✅ 오늘 처리할 항목 목록이 표시되면 다음 단계로 진행하세요.

STEP 4 / 6

열기 버튼 → 학생 수만큼 출력

1. 열기 버튼을 클릭해 파일을 엽니다.

2. 학생 수만큼 출력합니다.

✅ 출력이 정상적으로 끝났으면 다음 단계로 진행하세요.
4 / 6
STEP 5 / 6

완료 버튼 클릭 → 초록 배경 확인

1. 출력 완료 후 반드시 완료 버튼을 클릭합니다.

⚠️ 완료 클릭 안 하면 다음 조교가 중복 출력합니다.
✅ 항목이 초록 배경으로 바뀌면 완료된 것입니다.

STEP 1 / 3

자료 열람 > 폴더 탐색 → 목록에 추가

1. 좌측 트리에서 단원을 선택하고 우측 파일 목록을 확인합니다.

2. 필요한 파일을 체크한 후 목록에 추가 버튼을 클릭합니다.

✅ 선택한 파일이 목록에 담기면 다음 단계로 진행하세요.
1 / 3
STEP 2 / 3

내 목록/계획 탭 → 목록 이름 지정

1. 내 목록/계획 탭을 클릭합니다.

2. 저장된 목록을 선택한 후 이름 변경 버튼으로 명확한 이름을 부여합니다.

✅ 목록 이름이 알아보기 쉽게 바뀌었으면 다음 단계로 진행하세요.
2 / 3
STEP 3 / 3

계획 등록 (반·학생·날짜 선택 → 등록)

1. 목록 선택 드롭다운에서 목록을 선택합니다.

2. 반과 학생을 선택합니다 (N명 선택됨 확인).

3. 수업 예정일을 선택합니다.

4. N명 × M개 파일 = Task 수를 확인한 후 계획 등록 (N개) 버튼을 클릭합니다.

✅ 계획 등록 완료 메시지가 뜨면 조교에게 동기화됩니다.
3 / 3

  • ✅ 목록에 알아보기 쉬운 이름 지정
